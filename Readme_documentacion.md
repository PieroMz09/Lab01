# 📚 README - Documentación del Proyecto

## 📌 Descripción

Este proyecto es una plantilla web de e-commerce diseñada con HTML, CSS, JavaScript y PHP básico para formularios.

---

## 🧱 Estructura del proyecto

```
/eiser
│── index.html
│── category.html
│── single-product.html
│── cart.html
│── checkout.html
│── blog.html
│── contact.html
│── contact_process.php
│
├── css/
├── js/
├── img/
└── vendor/
```

---

## 🎨 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (vanilla)
* PHP (procesamiento de formularios)

---

## 📄 Páginas principales

### 🏠 Home

* `index.html`
* Presenta productos destacados

### 🛍️ Catálogo

* `category.html`
* Lista de productos

### 📦 Producto

* `single-product.html`
* Detalle del producto

### 🛒 Carrito

* `cart.html`

### 💳 Checkout

* `checkout.html`

### 📬 Contacto

* `contact.html`
* Usa `contact_process.php`

---

## 📬 Formulario de contacto

Archivo backend:

```
contact_process.php
```

Función:

* Procesa datos del formulario
* Puede ser extendido para enviar emails

---

## 🖼️ Recursos

* Imágenes en `/img`
* Scripts en `/js`
* Estilos en `/css`

---

## ⚙️ Configuración

### Requisitos:

* Navegador web moderno
* Servidor local para PHP:

  * XAMPP / WAMP / LAMP

### Ejecución:

1. Colocar proyecto en carpeta del servidor (htdocs)
2. Ejecutar Apache
3. Abrir en navegador:

```
http://localhost/eiser/
```
